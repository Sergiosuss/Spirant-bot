# 🎭 SPIRANT BOT - ПЛАТЕЖНАЯ СИСТЕМА

## 📦 ПОЛНЫЙ ПАКЕТ РЕШЕНИЯ

**Автоматическое обновление Google Sheets при поступлении платежей от ERIP (ipay.by)**

Статус: ✅ **ПОЛНОСТЬЮ ГОТОВО К ИСПОЛЬЗОВАНИЮ**

---

## 🚀 БЫСТРЫЙ СТАРТ (5 шагов)

### 1️⃣ Загрузи файлы
```bash
# Все файлы в /outputs/:
- spirant_payment_processor_final.py  # Обработка платежей
- email_payment_reader.py             # Чтение email
- main_with_payments.py               # Главный файл бота
- requirements.txt                    # Зависимости
- .env.example                        # Пример конфига
```

### 2️⃣ Установи зависимости
```bash
pip install -r requirements.txt
```

### 3️⃣ Настрой переменные окружения
```bash
cp .env.example .env
# Заполни в .env:
# - BOT_TOKEN (от BotFather)
# - ADMIN_CHAT_ID (твой Telegram ID)
# - GMAIL_APP_PASSWORD (от Gmail)
# - google_creds.json (от Google Cloud)
```

### 4️⃣ Загрузи Google credentials
- Скачай JSON из Google Cloud Console
- Сохрани как `google_creds.json`
- Предоставь доступ боту к таблице "Заявки"

### 5️⃣ Запусти бота
```bash
python main_with_payments.py
```

## 🔄 КАК ЭТО РАБОТАЕТ

```
EMAIL (ipay@ipay.by)
    ↓
GMAIL IMAP (zhmykhtv@gmail.com)
    ↓
PARSE TXT FILE (регулярные выражения)
    ↓
FIND STUDENT (по договору или ФИО)
    ↓
GOOGLE SHEETS API (обновить ячейку)
    ↓
TELEGRAM NOTIFICATION (уведомить админа)
```

### Пример:

**Входящее письмо от ipay@ipay.by:**
```
Номер договора (заказа) : ТС22044
ФИО                     : Червоная Марина Владимировна
Сумма                   : 184.00
Оплачено (дата/время)   : 2026.05.09 15:25:46
```

**Результат в Google Sheets:**
```
Студент: Червоная М.В. (строка 35)
Месяц: Май (столбец Y)
Ячейка: Y35 = 184.00
```

**Уведомление админу:**
```
✅ Платеж обновлен!
📄 ТС22044
👤 Червоная М.В.
💰 184.00 руб.
📍 Y35
```

## 📋 ЧТО НУЖНО ЗНАТЬ

### Google Sheets
- **Таблица:** "Заявки"
- **Лист:** "25/26" (основной рабочий лист)
- **Студент найдется по:** Договору (J) или ФИО (B)
- **Платежи пишутся в:** Столбцы Q-Y (сентябрь-май)

### Email
- **Отправитель:** ipay@ipay.by
- **Формат:** TXT файл в письме
- **Кодировка:** CP1251 (Windows Cyrillic)
- **Проверка:** Раз в час автоматически

### Telegram
- **Админ получает:** Уведомления о каждом платеже
- **Может проверить:** Статус вручную (`/sync_payments`)
- **Может отладить:** `/check_sheet`, `/check_email`

## 🐛 РЕШЕНИЕ ПРОБЛЕМ

### Бот не запускается?
→ Проверь `BOT_TOKEN` в .env

### Платежи не обновляются?
→ Используй `/sync_payments` и посмотри логи

### Ошибка "Gmail credentials invalid"?
→ Убедись что используешь **App Password**, не обычный пароль

### Не видит студента?
→ Проверь что договор в столбце J или ФИО в столбце B
